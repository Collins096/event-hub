Event Hub is a simple semantic HTML webpage designed to showcase upcoming and past events.  
It demonstrates the use of HTML5 elements such as `<header>`, `<main>`, `<section>`, `<article>`, and `<img>` to create a structured and accessible page layout.

## Project Overview

The webpage includes:

- A header with site title and navigation
- An **Upcoming Events** section with event titles and descriptions
- A **Past Events** section with event titles, descriptions, dates, and images
- Semantic HTML elements for better structure and accessibility
